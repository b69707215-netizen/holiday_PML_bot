# 🤖 Бот для автоматизации школы

Telegram-бот для автоматизации процессов формирования приказов, учета отпускных дней сотрудников и коммуникации между учителями и секретарем.

## 📋 Возможности

### 👨‍🏫 Для учителей:
- ✅ Регистрация с указанием ФИО и телефона
- 📊 Просмотр оставшихся дней отпуска
- 📝 Подача заявок на отпуск
- 📄 Автоматическое получение приказов после одобрения

### 👩‍💼 Для секретаря:
- 📨 Уведомления о новых заявках на отпуск
- ✅/❌ Одобрение или отклонение заявок
- 📄 Автоматическая генерация приказов (Word .docx)
- 👥 Просмотр списка сотрудников
- 📊 Мониторинг отпусков
- 📄 Создание приказов по шаблонам

## 🚀 Установка

1. **Клонируйте репозиторий:**
```bash
git clone <repository-url>
cd "ТГ бота для автоматизацый школы"
```

2. **Создайте виртуальное окружение:**
```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

3. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

4. **Создайте файл .env:**
```bash
cp .env.example .env
```

5. **Настройте переменные окружения в .env:**
```
BOT_TOKEN=your_telegram_bot_token_here
DATABASE_URL=sqlite:///school_bot.db
SECRETARY_TELEGRAM_ID=123456789
```

- `BOT_TOKEN` — получите у [@BotFather](https://t.me/BotFather)
- `SECRETARY_TEGRAM_ID` — Telegram ID секретаря (опционально, для уведомлений)

## 🎮 Запуск

```bash
python main.py
```

## 🛠 Технологии

- **aiogram 3.x** — фреймворк для Telegram Bot API
- **SQLAlchemy 2.x** — ORM для работы с БД
- **aiosqlite** — асинхронный драйвер SQLite
- **python-docx** — работа с Word документами
- **python-dotenv** — управление переменными окружения

## 📊 База данных

### Таблицы:
- **users** — сотрудники (учителя и секретарь)
- **vacations** — заявки на отпуск
- **orders** — история созданных приказов
